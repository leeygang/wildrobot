# Kinematics (Milestone 0)

This directory provides bounded nominal IK utilities for the locomotion stack.

Milestone `v0.19.2` includes:

- `leg_ik.py`: simple sagittal 2-link leg IK with reach clipping and reachability reporting

Deferred:

- full 3D whole-leg IK
- yaw/frontal-plane coupling
- optimization-based IK variants
